# Media Routing

- `examples/gltf/src/app.rs`: GLTF model and environment map loading.
- `examples/splat/src/app.rs`: splat rendering and mixed 3D assets.
- `examples/charts/src/app.rs`: chart widgets including `Chart3D`.
- `examples/vector/src/app.rs`: `Svg` widget and custom `DrawVector` widget rendering.
- `examples/uizoo/src/tab_image.rs`: image fit modes and image widgets.
- `examples/uizoo/src/tab_video.rs`: video widget usage.
- `examples/map/src/app.rs`: map widget usage (`maps` feature).
- `examples/pdf/src/app.rs`: PDF viewer usage (`pdf` feature).
- `examples/splash/src/app.rs`: integrated examples for map/svg/image under one shell.
