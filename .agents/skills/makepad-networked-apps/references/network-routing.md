# Network Routing

- `examples/aichat/src/app.rs`: chat UI + backend selection + streamed/rich response rendering.
- `examples/comfyui/src/app.rs`: scripted network/file orchestration with preview updates.
- `examples/automate/src/app.rs`: UDP/DMX control surface with timers and control widgets.
- `examples/arracing/src/app.rs`: gamepad/wheel input loop and UDP payload emission.
- `examples/teamtalk/src/app.rs`: larger communication-oriented app shell.
- `examples/git/src/app.rs`: async-ish data list behavior and action-driven updates.
