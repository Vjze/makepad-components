# UI Routing

Use this index to choose the smallest fitting UI template.

- `examples/counter/src/app.rs`: minimal app bootstrap and action handling.
- `examples/todo/src/app.rs`: stateful list CRUD in `script_mod!`.
- `examples/text_input/src/app.rs`: return handling and keyboard/input modes.
- `examples/uizoo/src/app.rs`: multi-tab Dock shell and modular script registration.
- `examples/uizoo/src/tab_*.rs`: isolated widget-specific demos.
- `examples/uizoo/src/demofiletree.rs`: FileTree data model + rendering.
- `examples/splash/src/app.rs`: large integrated UI with Dock, PortalList, FileTree, custom widgets.
- `examples/git/src/app.rs`: data-backed PortalList wrapper widget.
- `examples/aichat/src/app.rs`: rich markdown + code/math + chat list structure.
